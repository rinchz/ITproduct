# ASCII Art Converter

Небольшой скрипт `convert.py`, который запрашивает строку у пользователя, преобразует символы в простые ASCII-арт буквы/цифры по предопределённым шаблонам и выводит результат в консоль. Сторонние библиотеки не используются.

## Требования

- Python 3.9+ (подойдёт и новее)

## Запуск

Из корня проекта:

```bash
python3 convert.py
```

Далее введите строку и нажмите Enter — скрипт выведет ASCII-арт.

## Примечания

- Поддерживаются: латинские буквы `A-Z`, цифры `0-9`, пробел.
- Для неподдерживаемых символов выводится `?`.
